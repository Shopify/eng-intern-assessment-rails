
eng-intern-assessment-rails

This Ruby on Rails application provides a robust system for managing articles. It includes functionality for creating, uodating, deleting, and searching articles. Below is an overview of key features and instructions for setting up and using the application.

Features
Article Listing:

The index action retrieves and displays all articles.
Article Creation:

The new action initializes a new article for creation.
The create action attempts to save a new article, redirecting to the index on success or re-rendering the form on errors.

Article Editing:
The edit action renders the article edit form.
The update action attempts to update the article, redirecting to the index on success.

Article Viewing:
The show action renders individual article details

Article Deletion:
The destroy action deletes the specified article and redirects to the index.

Article Search:
The search action searches for articles based on the provided query, rendering the index view.
